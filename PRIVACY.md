# 隱私權政策 (Privacy Policy)

**Short URL Extension**  
**最後更新日期：2025年9月9日**

## 概述

Short URL Extension（以下簡稱「本擴展」）致力於保護用戶的隱私權。本隱私權政策說明我們如何收集、使用和保護您的資訊。

## 資料收集

### 我們不收集的資料

- **個人身份資訊**：我們不收集您的姓名、電子郵件地址、電話號碼或其他個人識別資訊
- **瀏覽歷史**：我們不追蹤或記錄您的瀏覽習慣
- **個人檔案**：我們不建立用戶個人檔案或行為分析
- **第三方資料**：我們不與第三方分享任何用戶資料

### 本地存儲的資料

本擴展僅在您的本地瀏覽器中存儲以下資料：

1. **API 設定**
   - Base URL（縮網址服務的 API 端點）
   - Authorization Token（您提供的 API 憑證）

2. **縮短網址歷史記錄**
   - 原始網址
   - 縮短後的網址
   - 頁面標題
   - 創建時間戳

## 資料使用

### 本地功能

- **API 設定**：用於向您指定的縮網址服務發送請求
- **歷史記錄**：用於在擴展界面中顯示您的縮短網址歷史，方便您管理和重複使用

### 外部 API 通訊

本擴展會向您配置的縮網址服務（如 aiurl.tw、glsoft.ai）發送 HTTPS 請求，包含：

- 要縮短的網址
- 您的 API 憑證

這些請求受到各服務提供商的隱私政策約束，請查閱相關服務的隱私政策。

## 資料安全

### 本地安全

- 所有資料僅存儲在您的瀏覽器本地存儲中
- 使用 Chrome 的 `chrome.storage.sync` API，資料可能會同步到您登入的其他 Chrome 瀏覽器
- Authorization Token 在設定頁面中以密碼形式顯示，防止意外洩露

### 傳輸安全

- 所有 API 請求均使用 HTTPS 加密傳輸
- 不會向未經授權的第三方發送任何資料

## 權限使用

本擴展請求的權限僅用於核心功能：

- **contextMenus**：在右鍵選單中添加縮短網址選項
- **storage**：在本地存儲設定和歷史記錄
- **activeTab**：獲取當前頁面 URL 和標題
- **clipboardWrite**：將縮短的網址複製到剪貼板
- **notifications**：顯示操作結果通知
- **scripting**：在頁面環境中執行剪貼板操作
- **host_permissions**：向指定的縮網址服務發送 API 請求

## 資料控制

### 用戶控制

您可以隨時：

- 查看所有存儲的歷史記錄
- 清除全部歷史記錄
- 修改或刪除 API 設定
- 導出歷史記錄資料
- 卸載擴展以清除所有本地資料

### 資料保留

- 歷史記錄：保留最近 50 筆記錄，超過會自動刪除舊記錄
- API 設定：持續保存直到您手動修改或刪除
- 卸載擴展時，所有本地資料會被完全清除

## 第三方服務

本擴展可能與以下第三方縮網址服務互動：

- aiurl.tw
- glsoft.ai
- 其他用戶自定義的服務

這些服務有各自的隱私政策，我們建議您查閱相關政策以了解它們如何處理您的資料。

## 兒童隱私

本擴展不專門針對 13 歲以下兒童設計，也不會故意收集此年齡層的個人資訊。

## 隱私政策變更

如果我們更新此隱私政策，會在擴展中或相關網站上發布新版本。重大變更時，我們會提供更明顯的通知。

本隱私政策受到相關地區的隱私保護法律約束，包括但不限於歐盟 GDPR、加州 CCPA 等規範。
