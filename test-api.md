# API 測試說明

## 支援的 API 欄位

根據原始代碼分析，API 支援以下欄位：

### 必要欄位
- `url` (string) - 要縮短的網址

### 可選欄位
- `slug` (string) - 自定義短網址後綴
- `comment` (string) - 評論/註解
- `expiration` (string) - 過期時間

## API 請求範例

```json
{
  "url": "https://example.com/very/long/url",
  "slug": "custom123",
  "comment": "這是測試用的網址",
  "expiration": "7d"
}
```

## 過期時間選項

- `1h` - 1小時後過期
- `24h` - 24小時後過期
- `7d` - 7天後過期
- `30d` - 30天後過期
- `1y` - 1年後過期
- 空值 - 永不過期

## 使用方式

1. 在自定義網址區塊輸入要縮短的網址
2. 可選：輸入評論
3. 可選：輸入自定義短網址代碼
4. 可選：選擇過期時間
5. 點擊「縮短自定義網址」按鈕

系統會自動將這些資訊包含在 API 請求中，並在歷史記錄中顯示額外資訊。
